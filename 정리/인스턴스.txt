핵심 
클래스는 객체를 만들기 위한거 메모리에 올리기 위한 변수와 메서드를 만드는 것 

class 클래스이름 { 변수들 ,메서드 }

new 클래스이름();->이렇게 만들면 참조값 (주소값)을 만든다 참조변수 저장 

참조변수.변수이름 -> 참조하는법
참조변수.메서드이름 ->메서드 호출

내가 써야함수는 뭐냐 메서드는 무엇이냐 를 정하는 것이 중요함 

사과장수와 	구매자 --. 객체

인스턴스는  생성이 되고 소멸될때까지 길다 

ㅎㅐㄱ심은  객체를 만들어 쓰자 

구조체는 타입이 서로다른 데이터 집합->파이썬- 리스트 ,자바스크립트-배열

클래스메서드는 메서드 이름으로 호출한다,
클래스이름.메서드이름();을 호출한다

인스턴스를 생성해야 인스턴스메소드를 호출 할 수있다,

스태틱은 static이 붙어있는 메서드나 변수는 프로그햄시작 코드가 메모리로 로드를 할때 생성

인스턴스를 생성 할때 사용가능 

인스턴스는 생성하고 사용한다


클래스를 정의하는 방법 ->인스턴스 생성->참조변수를 이용해서 인스턴스의 변수와 메소드 호훌
-> static 클래스 변수 클래스 메소드 

생성자(constructor)
-> 변수 초기화
